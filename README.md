# MERN Authentication Backend

## 📌 Project Overview
This project is a backend authentication system built using Node.js, Express, and MySQL. It includes user registration, login functionality, and secure access using JWT authentication.

---

## 🚀 Tech Stack
- Node.js
- Express.js
- MySQL
- JWT (JSON Web Token)
- bcrypt (Password Hashing)

---

## 🔑 Features
- User Registration with hashed passwords
- User Login with JWT authentication
- Protected Dashboard route
- Input validation
- Error handling
- Environment variables for security
